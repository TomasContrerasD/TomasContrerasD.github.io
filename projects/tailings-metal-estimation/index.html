Estimation of Heavy Metal Concentrations in Tailings Dams

MSc thesis project using multispectral satellite imagery and machine learning to estimate heavy metal concentrations in tailings deposits in Chile.

Overview

This project developed supervised machine learning models to estimate metal content in Chilean tailings deposits using geospatial, environmental, industrial, and multispectral satellite data.

Highlights

  • Developed supervised regression models for metal concentration estimation.
  • Tested and compared multiple modelling approaches.
  • Improved predictive performance by approximately 10%.
  • Worked with heterogeneous geospatial and environmental datasets.
